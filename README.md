# 📊 Blinkit Data Analysis Dashboard (Power BI)

## 📌 Overview

This project presents an interactive **Blinkit Data Analysis Dashboard** built using Power BI to analyze sales performance, product characteristics, and outlet distribution. The dashboard helps in understanding customer preferences, item visibility, and outlet performance across different locations.

---

## 🎯 Objectives

* Analyze product performance based on item type and fat content
* Understand outlet distribution by size, type, and location
* Evaluate item visibility and its impact on sales
* Identify trends in outlet establishment over time

---

## 📂 Dataset

The dataset used contains information such as:

* Item Type
* Item Fat Content
* Item Visibility
* Item Weight
* Outlet Size & Type
* Outlet Location Tier
* Outlet Establishment Year
* Sales & Ratings

---

## 📈 Dashboard Features

### 🔹 1. Rating by Item Type

* Donut chart showing ratings distribution across item categories:

  * Fruits & Vegetables
  * Snack Foods
  * Household
  * Frozen Foods
  * Dairy

### 🔹 2. Outlet Size Distribution

* Pie chart representing:

  * Small
  * Medium
  * High outlet sizes

### 🔹 3. Item Visibility Analysis

* Highlights total item visibility and its contribution to performance

### 🔹 4. Outlet Location Type

* Bar chart showing performance across:

  * Tier 1
  * Tier 2
  * Tier 3 locations

### 🔹 5. Item Weight by Fat Content

* Area chart analyzing:

  * Low Fat
  * Regular
* Helps understand product health trends

### 🔹 6. Rating by Outlet Type

* Donut chart comparing ratings across:

  * Supermarket Type1
  * Grocery Store
  * Supermarket Type2 & Type3

### 🔹 7. Outlet Establishment Year

* Line/area chart showing trends of outlet growth over time

### 🔹 8. Outlet Type Performance

* Bar chart comparing sales/performance by outlet types

---

## 🛠️ Tools & Technologies

* **Power BI** – Data visualization
* **Excel / CSV Dataset** – Data source
* **DAX (Data Analysis Expressions)** – Measures and calculations

---

## 📊 Key Insights

* Supermarket Type1 outlets dominate in performance
* Tier 3 locations show higher sales compared to Tier 1 & Tier 2
* Low-fat items have significant demand
* Item visibility plays a crucial role in product performance
